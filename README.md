## Getting Started

### Prerequisites
You must have Docker installed. [Download](https://www.docker.com/community-edition#/download)

### Build
```bash
docker build -t sonarwhal .
```

### Testing
```bash
docker runsonarwhal
```

### Pull
```bash
docker pull willfarrell/sonarwhal
```

## Deployment
```bash
docker run willfarrell/sonarwhal https://example.com
```
